# USPIRE ERP

## Purpose
Governance-first ERP built from the USPIRE ERP PRD (Phase 0 foundations first).

## Non-negotiables
- Workflow-governed processes
- Immutable audit trail
- DoA + SoD enforcement
- SOP enforcement
- Overrides logged and reviewable

## Planned tech stack
- Frontend: React + TypeScript
- Backend: NestJS + TypeScript
- Database: PostgreSQL

## Project structure (placeholder)
- `/docs`
- `/backend`
- `/frontend`

## How to run locally (placeholders)
```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
npm install
npm run start:dev
```
